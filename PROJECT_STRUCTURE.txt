Key Technologies:
  Backend:     FastAPI, Python 3.11+, uvicorn
  Frontend:    HTML, Tailwind CSS, HTMX, Tom-Select
  AI/ML:       Mistral Pixtral-12B (OCR), Claude Sonnet 4 (extraction)
  Storage:     Google Sheets (data), Google Cloud Storage (images)
  Deployment:  Docker, Docker Compose

API Endpoints:
  GET  /                      Landing page
  GET  /manual                Manual input form
  GET  /ocr                   OCR bulk upload
  GET  /api/stores            Get stores (filtered by area)
  POST /api/manual-submit     Submit manual entry
  POST /api/ocr-process       Process images with OCR
  POST /api/ocr-submit        Submit OCR bulk entries
  GET  /health                Health check

Data Flow:
  Manual:  User Form → Validation → Google Sheets
  OCR:     Images → GCS Upload → Mistral OCR → Claude Extract → 
           SKU Converter → Preview → User Edit → Google Sheets
